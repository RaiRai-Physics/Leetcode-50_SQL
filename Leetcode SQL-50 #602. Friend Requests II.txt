Leetcode SQL-50: #602. Friend Requests II: Who Has the Most Friends

Question Link: https://leetcode.com/problems/friend-requests-ii-who-has-the-most-friends/?envType=study-plan-v2&envId=top-sql-50

# Write your MySQL query statement below
select
id,
count(*) as num
from (
select requester_id as id from requestaccepted
union all
select accepter_id as id from requestaccepted
) t
group by id
order by num desc
limit 1;
