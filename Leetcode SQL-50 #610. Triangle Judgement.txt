Leetcode SQL-50: #610. Triangle Judgement

Question Link: https://leetcode.com/problems/triangle-judgement/?envType=study-plan-v2&envId=top-sql-50

/* Write your T-SQL query statement below */
select
x, y, z,
case
when x + y > z and x + z > y and y + z > x then 'Yes'
else 'No'
end as triangle
from Triangle;
