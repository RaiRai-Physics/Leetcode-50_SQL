Leetcode SQL-50; #1633. Percentage of Users Attended a Contest

Question Link: https://leetcode.com/problems/percentage-of-users-attended-a-contest/?envType=study-plan-v2&envId=top-sql-50

# Write your MySQL query statement below
SELECT
r.contest_id,
round(
count(DISTINCT r.user_id) * 100.0 / (SELECT COUNT(*) FROM Users),2) AS percentage
FROM Register r
GROUP BY r.contest_id
ORDER BY percentage DESC, contest_id ASC;
